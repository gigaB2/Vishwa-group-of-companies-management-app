# Web Application Project

An interactive web application built with HTML, JavaScript, and Vanilla CSS.

## Features

- Dynamic UI components and interactive modules
- Modular structure with CSS and JavaScript separations
- Clean design and styling setup

## Getting Started

1. Clone this repository:
   ```bash
   git clone <repository-url>
   ```
2. Open [`index.html`](file:///C:/Users/Admin/Desktop/work/main/index.html) in your browser or serve using a local server.

## Project Structure

- `index.html` - Main application entry point
- `sidebar.css` & `sidebar.js` - Main layout styles and functionality
- `modules/` - Core module implementations
- `assets/` - Project static assets
- `.env.example` - Example environment configuration template
